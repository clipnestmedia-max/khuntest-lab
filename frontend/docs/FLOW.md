# KhunTest Lab Flow

## Patient Flow
1. Register/Login
2. Select test/package
3. Choose Home Collection or Centre Visit
4. Pay online/cash
5. Receive confirmation
6. Sample collection
7. Admin uploads report
8. Patient downloads PDF report

## Admin Flow
1. Login
2. View new bookings
3. Assign home collection staff
4. Update booking status
5. Upload PDF report
6. Notify patient by WhatsApp/SMS
